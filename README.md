---- About ----

nim-csv2json is a Nim module for converting data from CSV into its JSON representation.

Example input (CSV):
This,is,a
test,1,2
3,hello,world

Example output (JSON):
{"data": [
    ["This", "is", "a"],
    ["test", "1", "2"],
    ["3", "hello", "world"]
]}

See the documentation.html file for usage and more info.
